# 🎬 IMDB Sentiment Analysis (LSTM + FastAPI + Streamlit)

## 🚀 Project Overview
This project performs **sentiment analysis** on IMDB movie reviews using a Deep Learning model (LSTM).

It is a complete **end-to-end ML project**:
- Data preprocessing
- Model training
- API deployment (FastAPI)
- UI (Streamlit)

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- LSTM (Deep Learning)
- FastAPI (Backend API)
- Streamlit (Frontend UI)
- Scikit-learn
